#  `ft [k] [v ...]` flag tags


```
Usage: f [?|-name| name] [@addr] # Manage flagzones
```

- `ft tag strcpy strlen ...` set words for the 'string' tag
- `ft tag`                   get offsets of all matching flags
- `ft`                        list all tags
- `ftn tag`                   get matching flagnames fot given tag
