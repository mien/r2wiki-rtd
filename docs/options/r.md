#  `r` Resize file


```text
Usage: r[+-][ size]Resize file
```


- `r` display file size
- `r size` expand or truncate file to given size
- `r-num` remove num bytes, move following data down
- `r+num` insert num bytes, move following data up
- `rm [file]` remove file
- `rh` show file size in human format
- `r2 [file]` launch r2
