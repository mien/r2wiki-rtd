#  **`dmp[?] <address> <size> <perms>`** Change page at <address> with <size>, protection <perms> (rwx)


```text
Usage: dmp Change page permissions
```


- `**dmp [addr] [size] [perms]**` Change permissions
- `**dmp [perms]**` ** ** Change dbg.map permissions
